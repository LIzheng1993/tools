- 如仅作为消费者或生产者，直接使用consumer.php或producer.php文件并安装扩展即可。

- PHP要安装rdkafka扩展，而rdkafka又依赖librdkafka，因此你需要安装rdkafka和librdkafka，之后就可以与kafka服务器交互了。
